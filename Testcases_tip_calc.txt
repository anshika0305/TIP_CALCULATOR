UI and Functional test cases for this Tip Calculator:

UI testcases - 
1.  Verify if the web application is loading properly.
2.  Verify if the labelling of the form is done correctly.
3.  Ensure that the fields of the form accepts only Numerical values.
4.  Verify the response when no or partial inputs are provided.
5.  Verify if the button is functional.
6.  Verify if all inputs are provided correctly, the button works.
7.  Verify the result display area is reset default when first visited the page.
8.  Verify the result is properly displayed.
9.  Verify if any pop-ups appear.
10. Verify the resuability of the tip calculator.
11. Verify if the UI of the calculator is user friendly or not.
12. Verify if the slider in the calc is sliding perfectly or not.
13. Ensure if the colours used for background and placeholders are pleasing or not.
14. Verify if the visibility of the application is clear or not.


Functional Testcases - 
15. Verify if all inputs are correctly parsed, and if they accept correct datatype. 
16. Verify if the output calculated on partial input, it provides proper response.
17. Verify if all inputs are provided, the output displayed is correct.
18. Verify if the format of output is correct. 
19. Verify if the tip-calculator is reused, the old result value does not interfere with the new result value.
20. Verify if the Amount is correctly divided among the no. of persons entered by user.
